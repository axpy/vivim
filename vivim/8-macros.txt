*Macros*

q + register - start recording macro (qa, q1)
q + register uppercase - append macro commands to existing one
@ + register - run macro
@ + @ - run recently executed macro

.,$ normal @register - repeat macro @register for lines from '.' to '$' (10,25 normal @a)

d + /delete-to-term - delete to `delete-to-term`

"register p - paste data of macro to see/modify
"register C - update register macro

Ctrl-v + ENTER/Escape... - to insert special carrot (^M, ^[...) symbols inside for macro)
